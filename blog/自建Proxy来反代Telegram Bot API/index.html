自建Proxy来反代Telegram Bot API

2023年1月2日 · 阅读需 1 分钟
时光's
For Freedom

前提：

  1. 有一台可以科学上网的服务器/vps/小鸡/软路由
  2. Nginx+PHP环境

Demo：

  • 传送门

Github：

  • 传送门

教程：

话不多说，开搞，此处以宝塔面板为例

  • 新建一个站点，并将你的域名解析到此站点，PHP版本我个人设置的为7.4，Github中提示PHP版本要求≥5.6即可，将Github中的index.php和tgproxy.php放入站点目录中，如图所示，具体步骤不再详细阐述。

  • 设置站点ssl，并开启强制https
  • 设置伪静态规则
rewrite ^/bot.*$ /tgproxy.php last;

  • 大功告成，此时访问你刚才所添加的站点的域名，即可

调用方法：

https://tg-bot.0x23.cf/botxxxxxxx:xxxxxxxxxxxxxxx/sendMessage?chat_id=xxxxxx&text=xxxxx
